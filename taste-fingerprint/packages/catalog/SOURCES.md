# 3D Model Sources for Furniture Catalog

## Free 3D Model Repositories (CC0 / Public Domain)

### 1. **Poly Haven** (polyhaven.com)
- License: CC0 (Public Domain)
- Best for: High-quality furniture, realistic models
- Format: GLB/GLTF available
- Categories: Furniture, props, decorative items

### 2. **Sketchfab** (sketchfab.com)
- Filter by: "Downloadable" + "CC0" or "CC-BY"
- Format: GLB download available
- Categories: Furniture, lighting, art, decor
- Note: Always check individual licenses

### 3. **Three.js Examples** (threejs.org)
- License: MIT
- Format: GLB/GLTF
- Available models in: `/examples/models/gltf/`
- Good starter models for testing

### 4. **Quaternius** (quaternius.com)
- License: CC0
- Format: GLB available
- Style: Low-poly, stylized
- Good for: Performance-optimized models

### 5. **KayKit** (kaylousberg.com)
- License: CC0
- Format: GLB
- Style: Low-poly, cartoonish
- Good for: Lightweight models

### 6. **Free3D** (free3d.com)
- Mixed licenses (check each)
- Format: Various (GLB available)
- Categories: Extensive furniture library

### 7. **Kenney.nl** (kenney.nl)
- License: CC0
- Format: GLB available
- Style: Low-poly game assets
- Categories: Furniture packs available

## Model Requirements

- **Format**: GLB or GLTF 2.0
- **Size**: < 5MB per model (ideally < 2MB)
- **Poly Count**: < 50k triangles for performance
- **Textures**: Embedded or separate (PBR preferred)
- **Scale**: Real-world units (meters)
- **Origin**: Centered at base for furniture
